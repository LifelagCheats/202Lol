- **Cookie handling**
- **Response viewing**
- **File sending**
- **Interactive interface**

## Installation

**prerequisites**
- python 3.8 or later
- pip
- git

```bash
git clone https://github.com/LifelagCheats/202Lol
cd 202Lol
pip install -r requirements.txt
python3 main.py
```
